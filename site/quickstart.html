Quickstart

From pip install to a stopped agent in 60 seconds.

Four commands, no account, no API key, no dashboard. You end with a real trace file and a guard you can paste into your own agent loop.

pip install agentguard47
Local proof path

Prove it works before trusting it.

Run these in a clean Python environment. Everything is local and offline.

1

Verify the install

Confirms the SDK imports, writes a trace, and prints your next step.

agentguard doctor
2

Trip a guard

Runs an offline agent that overspends, loops, and retry-storms. Each guard stops it and writes JSONL.

agentguard demo
3

Read the local report

Cost, guard events, and a savings-ledger summary from the trace the demo just wrote.

agentguard report agentguard_demo_traces.jsonl
4

Copy a starter into your repo

Writes a minimal, runnable starter for your stack. Pick a framework below.

agentguard quickstart --framework raw --write

The real integration

It is this small.

A hard dollar ceiling on a plain agent loop. No provider lock-in.

import agentguard

tracer = agentguard.init(
    service="repo-coder",
    budget_usd=5.00,
    trace_file="traces.jsonl",
    local_only=True,
)

with tracer.trace("agent.run") as span:
    span.event("tool.call", data={"tool": "edit"})
    # BudgetExceeded is raised mid-run when the ceiling is hit.
