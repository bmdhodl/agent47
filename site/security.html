Security

AgentGuard is designed with security as a first principle. Zero dependencies means zero supply chain risk. Here's how we protect your data.

Zero-dependency SDKPure Python stdlib. No transitive dependencies, no supply chain risk. One package to audit.

Transport securityAll API traffic encrypted with TLS 1.3. No plaintext data in transit.

Data at restSupabase Postgres with AES-256 encryption. Your trace data is encrypted on disk.

API key securityKeys are hashed with SHA-256 before storage. Only the prefix is visible in the dashboard.

Data retentionAutomatic cleanup per plan tier: 7 days (Free), 30 days (Pro), 90 days (Team). No indefinite data hoarding.

Access isolationAll queries are scoped to your team. No cross-tenant data access is possible.

Open sourceThe SDK is MIT-licensed and available on GitHub. Audit the source code anytime.

Responsible disclosureFound a vulnerability? Email security@agentguard47.com. We respond within 48 hours.
