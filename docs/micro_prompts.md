# Micro Prompts (AI Coding Agent)

Use these as single-step prompts.

1. "Scan the repo and summarize what exists. Suggest the next smallest, testable improvement."
2. "Create a product requirements doc for an MVP observability SDK for multi-agent systems."
3. "Define the event schema for traces and update docs to reflect it."
4. "Implement a minimal tracing SDK with JSONL output and a unit test that verifies emitted events."
5. "Add a loop guard that detects repeated tool calls and raises a clear exception."
6. "Implement a budget guard for token/call limits with reset support."
7. "Create a recorder/replayer utility that logs request/response pairs and replays deterministically."
8. "Add a tiny CLI command to summarize trace files by kind and name."
9. "Write a README quickstart showing tracing, guard, and replay usage."
10. "Draft a go-to-market checklist for OSS launch and add it to docs."
11. "Identify one framework integration target and create a stub adapter file."
12. "Propose a minimal integration example and add it to docs/examples."
13. "Add CI-ready tests for tracing and guards."
14. "List the top three risks and add mitigations to docs."
