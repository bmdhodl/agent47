# Cofounder Inbox

Lightweight SDK-to-cofounder handoff for this public repo.

Rules:
- Keep this SDK-only.
- Do not put business-sensitive strategy, outreach, revenue, or private
  dashboard details here.
- Keep notes short and current.
- Prefer one rolling status file over an ever-growing log.
- Use this to surface blockers, shipped changes, and decisions the cofounder
  agent should carry into daily context refreshes.

Files:
- [`latest.md`](latest.md) - current SDK status for the cofounder agent
