# Cofounder Inbox

Durable SDK-to-cofounder handoff for this public repo.

Rules:
- Keep this SDK-only.
- Do not put business-sensitive strategy, outreach, revenue, or private
  dashboard details here.
- Append one short entry after each merged PR.
- Keep entries high-signal and newest first.
- Use this to surface blockers, shipped changes, and decisions the cofounder
  agent should carry into daily context refreshes.

Standard entry format:
- Date
- Agent
- What shipped
- Decisions made
- Blockers

Files:
- [`log.md`](log.md) - durable merged-PR log for the cofounder agent
